Fullstack WebGIS berbasis PostgreSQL

Peta fasilitas kesehatan yang lebih interaktif dan siap dikelola admin.

Website ini dirancang ulang sebagai sistem WebGIS fasilitas kesehatan Kabupaten Purworejo. Pengguna umum dapat melihat peta, mencari lokasi, memfilter jenis fasilitas, melihat statistik, dan membuat rute. Admin dapat login untuk mengelola data spasial melalui basis data PostgreSQL/PostGIS.

Fitur utama yang menjawab instruksi tugas

Struktur website dibuat seperti platform geospasial modern: ada landing page, peta interaktif, katalog data, halaman admin, serta dokumentasi pengembangan.

⌖

Peta Interaktif

Visualisasi batas wilayah dan titik fasilitas kesehatan dengan basemap, legenda, popup atribut, filter, pencarian, serta zoom otomatis.

▣

Basis Data Spasial

Struktur PostgreSQL/PostGIS disiapkan untuk menyimpan titik fasilitas dan batas wilayah dalam tipe geometri.

↗

Analisis Ringan

Tersedia statistik jumlah data, grafik jenis fasilitas, pencarian fasilitas terdekat, buffer radius, dan ekspor CSV.

◉

Login Admin

Admin masuk melalui halaman login untuk menambah, mengubah, dan menghapus data fasilitas kesehatan.

Alur sistem

Satu website untuk publik, admin, data, dan laporan.
